= `qtts --model kitten`

https://github.com/KittenML/KittenTTS

== Privacy

Pulls
link:https://hub.docker.com/r/quinngrier/qtts-kitten[docker.io/quinngrier/qtts-kitten]
if needed, then runs completely locally via
`docker run --network none`.

* `expr-voice-3-m`
* `expr-voice-4-f`
* `expr-voice-4-m`
* `expr-voice-5-f`
